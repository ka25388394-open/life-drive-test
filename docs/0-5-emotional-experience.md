# 0-5｜Emotional Experience Specification

# 《你靠什麼感覺自己正在活著？》

## V1 Emotional Experience Specification

> 本文件定義本產品希望帶給使用者的**情緒體驗**。
>
> 這不是工程文件。不包含 CSS、動畫秒數、Component、UI 實作。
>
> 情緒基準延續:0-0 語氣、0-3 結果文案、0-4 情緒關鍵字。本文件不改動任何既有規格,只把散落在各文件中的情緒意圖,收攏成一份對齊用的基準。

| 來源 | 承接的情緒 |

# 1. Product Identity｜情緒身份

在情緒層面,本產品是:

> **一次溫柔的自我回顧。**

而不是:

- 一份人格分析報告。
- 一個把人分類、貼標籤的工具。
- 一個要你「答對」的測驗。

情緒立場:

- 產品**陪在使用者身邊**,不站在上方評斷。
- 使用者不是「被測驗的對象」,而是「被陪伴的人」。
- 它回答的不是「你是誰」,而是:
  - 「你需要什麼。」
  - 「為什麼最近可能會這樣。」

# 2. Emotional Journey｜五階段情緒歷程

```
開始測驗 → 回答問題 → 結果揭曉 → 閱讀結果 → 離開網站
```

每一階段應感受到什麼、應避免什麼:

---

### 階段一｜開始測驗

- **希望感受到:** 安心、被邀請、好奇而不緊張。
- **心裡浮現:** 「這好像不會考我。」「我可以誠實一點。」
- **應避免:** 被評估的壓力、要「表現好」的緊張、像要開始一場考試。

---

### 階段二｜回答問題

- **希望感受到:** 輕鬆、像在跟自己對話、像聊天(承 0-0 / 0-1 設計原則)。
- **心裡浮現:** 「這題在問我真正想要的是什麼。」
- **體驗:** 憑直覺就能回答,不需要煩惱「正確答案」。每一題都像有生活畫面的小問句,而非心理學問卷。
- **應避免:** 認知負荷、被審問感、猶豫對錯、覺得自己被拆解分析。

---

### 階段三｜結果揭曉

- **希望感受到:** 期待、被慎重對待、一點點「啊……」的觸動。
- **體驗:** 結果像翻開書的下一頁,慢慢揭開;而不是被丟出一張報告。
- **應避免:** 被下定論的瞬間、被貼上標籤的斷然感、被歸檔的冰冷。

---

### 階段四｜閱讀結果

- **希望感受到:** 被理解、被接住;浮現「原來我是這樣」「我好像更理解自己了」。
- **體驗:** 使用者感覺「有人懂我最近為什麼疲憊、為什麼空空的」,而不是「我被分析了」。文案像陪伴(承 0-3),讓人想慢慢讀完。
- **應避免:** 診斷、說教、評價、被分類;讀起來像報告或健康檢查結論。

---

### 階段五｜離開網站

- **希望感受到:** 平靜、被安慰、有一點希望、想對自己溫柔一點。
- **帶走的:** 不是一個標籤,而是一句「我好像知道自己需要什麼了」。承 0-3 結尾——願自己慢慢靠近那些讓自己有生命力的事物。
- **應避免:** 空虛離開、覺得「只是玩了個測驗」、被要求註冊/分享/導流的壓力。

---

# 3. User Feeling｜整體使用者感受

延續 0-4:

```
安靜
被理解
舒服
有希望
```

甚至,在某個句子上,有一點想哭的鬆動。🥹

整體閱讀感:

> 像深夜,安靜地翻完一本屬於自己的小書。

# 4. Emotional Keywords｜情緒關鍵字

承 0-4 §2(不新增、不偏移):

```
Gentle    溫柔
Warm      療癒

```

同一調性內的情緒動詞(用於對齊,不是新方向):

```
被理解、被陪伴、被接住、鬆一口氣、對自己溫柔一點。
```

---

# 5. Forbidden Feeling｜禁止的情緒

承 0-0 / 0-3 / 0-4 的「避免」清單:

- ❌ 被診斷。
- ❌ 被貼標籤 / 被分類。
- ❌ 被說教。
- ❌ 被評價 / 被打分數(分好壞、對錯)。
- ❌ 考試感 / 答錯的焦慮。
- ❌ 遊戲化的刺激、炫酷、花俏。
- ❌ 被推銷、被要求註冊或分享。

任何讓使用者覺得「我在被審視」的情緒,都不屬於本產品。

---

# 6. Final Emotional Goal｜最終情緒目標

使用者在離開時,心裡浮現:

> 「原來我是這樣。」
>
> 「我好像更理解自己了。」

並且,對自己多一分**溫柔與允許**。

產品成功的標準:

- 不是「準不準」。
- 而是「**被理解的感覺**」是否真的發生。

---

# 7. Core Design Sentence｜核心設計句

> 像深夜,安靜地翻完一本屬於自己的小書——
>
> 讀完之後,對自己更溫柔了一點。
